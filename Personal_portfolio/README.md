# 🌐 Tanvi Jivatode — Python Full Stack Developer Portfolio  

Tanvi Jivatode’s Portfolio is a Django-based web application built to showcase real-world projects, technical expertise, certifications, and development experience through a clean UI and scalable backend architecture.

---

🌐 **Live Demo:**  
🔗 https://tanvis-portfolio.onrender.com/

---

## 📌 Overview  

This project demonstrates practical full-stack development using Django, focusing on:

- Structured project architecture  
- Static asset management  
- Deployment workflow  
- Clean UI presentation  
- Backend configuration best practices  

The portfolio is deployed live on Render and structured following Django’s MVT (Model-View-Template) pattern.

---

## 🚀 Key Features  

- Professional landing page  
- Project showcase section  
- Certificates section  
- Resume display section  
- Organized static file handling  
- Django Admin panel integration  
- Scalable Django app structure  
- Production deployment on Render

## ▶️ Run Locally  

git clone https://github.com/tanvi2711/Python_Full_stack_Projects.git

cd Python_Full_stack_Projects/Personal_portfolio

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
